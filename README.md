## 🇬🇧 English <a name="english"></a>

A fun interactive game to ask "Will you be my girlfriend?". The "No" button is impossible to click!

### 🚀 Quick Start

1. **Open file** - Open `index.html` in your browser, no installation needed
2. **Select language** - Choose English, Azerbaijani, Turkish, or Russian
3. **IQ Test** - 15-second simple test, then the main question appears
4. **Main question** - "Will you be my girlfriend?" - Yes is green, No is red and runs away!

### ✨ Features

| Feature | Description |
| ------- | ----------- |
| 💕 **Question** | "Will you be my girlfriend?" |
| ✅ **Yes** | Green, triggers victory screen |
| ❌ **No** | Red, teleports away on hover |
| 🎞️ **Animations** | Hearts, running button |
| 📊 **Counter** | Tracks No attempts |
| 😾 **Angry cat** | Appears after 3 attempts |
| 📱 **Responsive** | Works on mobile and desktop |

## 🇷🇺 Русский <a name="russkiy"></a>

Игра для вопроса "Будешь ли ты моей девушкой?". Нажать кнопку "Нет" невозможно!

### 🚀 Быстрый старт

1. **Открой файл** - Открой `index.html` в браузере, установка не нужна
2. **Выбери язык** - Английский, Азербайджанский, Турецкий или Русский
3. **IQ тест** - 15-секундный простой тест, потом главный вопрос
4. **Главный вопрос** - "Будешь ли ты моей девушкой?" - Да зелёное, Нет убегает!

### ✨ Возможности

| Возможность | Описание |
| ----------- | -------- |
| 💕 **Вопрос** | "Будешь ли ты моей девушкой?" |
| ✅ **Да** | Зелёная, экран победы |
| ❌ **Нет** | Красная, убегает при наведении |
| 🎞️ **Анимация** | Сердечки, убегающая кнопка |
| 📊 **Счётчик** | Считает попытки нажать "Нет" |
| 😾 **Злой кот** | Появляется после 3 попыток |
| 📱 **Телефон** | Работает на компьютере и телефоне |
